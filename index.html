Q. 멋쟁이 사자처럼이 뭔가요?

멋사는 비전공자를 대상으로 한 프로그래밍 교육 동아리입니다.
우리는 원하는 웹 서비스를 직접 만들고, 런칭합니다!

아이디어뱅크 문과생: 만들고 싶은 서비스가 있는데 어떻게 만들지?ㅜㅜ
컴퓨터 전공생: 학교 수업은 열심히 들었지만 정작 만들 줄 아는 게 없어..
예술적 영감이 마구 떠오르는 미대생: 디자이너로서 창의력을 뿜어내고 싶어!

혹시 이런 생각들을 해본 적 있으신가요?
그렇다면 멋사로 오세요!

Q. (2020년 4월 예정)원데이 클래스에서는 무엇을 배우나요?

아래 주제에 대해 배웁니다.

타이타닉호 데이터 분석

python, pandas를 사용하여 타이타닉 생존자 데이터를 분석하여 나만의 결론을 도출해봅니다.

Q. 멋사에서는 무엇을 배우나요?

웹 프로그래밍을 위한 아래 세 가지를 배웁니다.

HTML/CSS/JavaScript

웹 페이지를 만들고, 꾸밀 수 있는 언어입니다. 코드카데미를 통해 기초를 배웁니다.

Python

멋사에서 사용하는 주 프로그래밍 언어입니다. 장고와 함께 웹 서비스에 다양한 기능을 만듭니다.

Django

파이썬과 함께 사용하는 프레임워크로 웹 서비스의 뼈대입니다.

멋사에 대해 자주 묻는 질문

학번, 나이제한이 있나요?

아니요. 없습니다! 숙명여자대학교 학생이라면 모두 가능합니다.

활동기간은 어떻게 되나요?

정규활동 기간은 1년입니다. 단, 중도 하차 시 멋사에서 제공하는 혜택을 일체 받으실 수 없습니다. (추후에도 불가)

비전공자여도 따라갈 수 있을까요?

네. 조건으로 걸린 강의만 수강하신다면 충분히 따라오실 수 있습니다.

합격했는데 사정이 생겨서 못하게 될 경우 다음 기수 재지원 가능한가요?

아니요. 이번기수에 합격하지 않은 지원자는 다음 기수 지원이 가능하지만, 이미 합격한 지원자는 제명회원으로 처리되어 다음 기수 지원이 불가능합니다.

경쟁률은 어떻게 되나요?

전 기수의 지원서 및 개인정보는 지원이 끝나는 즉시 폐기하기 때문에 경쟁률에 대해 언급드리기 어렵습니다.

무엇을 배우나요?

1학기는 중앙 강의에서 제공하는 창업 교육과 숙명여대 멋사에서 자체적으로 진행하는 Django 강의로 진행됩니다. 여름 방학에 중앙에서 내려오는 코딩 강의를 집중 수강한 뒤, 2학기에는 1학기에 결정한 아이디어를 실현하게 됩니다.
